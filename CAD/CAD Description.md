## Full Assembly Model (.STEP)

This directory contains the complete Titan Rover robotic arm assembly exported as a standard `.STEP` file for easy review, integration, and cross-platform compatibility. 

**Note on CAD Origins:** These models were originally designed and assembled using a Student Edition of SolidWorks. If you are importing this `.STEP` file back into a commercial version of SolidWorks, please be aware that it may trigger an educational watermark or flag.
